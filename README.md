## Project Overview

This repository contains my capstone project for the Open Source Software course. The project is an audit of **Git** - a distributed version control system created by Linus Torvalds in 2005. The audit covers Git's origin story, license analysis, philosophical values, Linux footprint, ecosystem mapping, and comparison with proprietary alternatives.

Along with the written report, this repository contains five shell scripts that demonstrate practical Linux skills and connect automation to open source philosophy.
